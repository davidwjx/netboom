Windows USB driver installtion instructios:

1. Temporary install the zip pacakge located at: /driver/zip folder
2. In order to create GUI driver installtion -  copy the /driver/xml/dpinst_gui.xml file to /driver/(nu3000_2endpoints or nu4000_1endpoint)/dpinst.xml
3. In order to create NOGUI driver installtion - copy the /driver/xml/dpinst_nogui.xml file to /driver/(nu3000_2endpoints or nu4000_1endpoint)/dpinst.xml
4. Open command Window. 
5. Activate the re-pack-files.cmd from the cmd window.
6. This application will output the InstallDriver.exe USB driver installtion
